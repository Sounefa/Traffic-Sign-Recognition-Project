Traffic Sign Recognition Project

A complete machine learning pipeline for classifying and detecting traffic signs. Built using AlexNet-based architectures and trained on the GTSDB dataset.

Launch Interactive Demo

Overview

This project implements a full traffic-sign recognition pipeline for self-driving cars. It includes model training, preprocessing, visualization, evaluation, and interpretability. The goal is to achieve real-time classification accuracy with lightweight architectures suitable for on-road deployment.

Dataset: GTSDB

The German Traffic Sign Detection Benchmark contains more than 900 images with over 1,200 labeled traffic signs. It includes bounding boxes and 43 classes of signs. Preprocessing includes resizing, normalization, augmentation, contrast enhancement, and noise reduction.

View Dataset Samples

Model Architecture (AlexNet-Based)

The model uses a modified AlexNet architecture optimized for traffic-sign classification. Improvements include:

• Depth-wise convolutions • Optional residual connections • Batch normalization • Accelerated training pipeline • High-resolution input support

Training Pipeline

Training includes:

• Data augmentation (flip, rotation, blur, color jitter)
• Cross-entropy loss
• Learning rate warmup + decay
• Batch size optimization
• Accuracy / loss visualizations
• Confusion matrix and prediction outputs

View Training Charts

Results & Performance

• Classification accuracy: 97–99% • Real-time inference performance < 12ms per frame • Strong detection capability using bounding-box assisted classification
