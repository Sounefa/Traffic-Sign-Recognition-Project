Traffic Sign Recognition Project

A complete machine learning pipeline for classifying and detecting traffic signs. Built using AlexNet-based architectures and trained on the GTSDB dataset.

Launch Interactive Demo

Overview

This project implements a full traffic-sign recognition pipeline for self-driving cars. It includes model training, preprocessing, visualization, evaluation, and interpretability. The goal is to achieve real-time classification accuracy suitable for deployment.

Dataset: GTSDB

The German Traffic Sign Detection Benchmark contains more than 900 images with over 1,200 labeled traffic signs across 43 classes.

Model Architecture

The model uses a modified AlexNet architecture optimized for traffic-sign classification, including batch normalization, enhanced convolutions, and optimized inference.

Training Pipeline

• Data augmentation (flip, rotation, blur, jitter)
• Cross-entropy loss
• Learning rate warmup + decay
• Accuracy & loss curves
• Confusion matrix and predictions

Results

• Accuracy: 97–99%
• Real-time inference (< 12ms/frame)
• Robust classification & detection
